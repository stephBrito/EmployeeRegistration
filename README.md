## Contexto
Essa API executa um simples CRUD para criação de novo colaborador.

## Criando imagem PostgresDB com Docker utilizando Compose

- Entrar no diretório raiz onde foi baixada a aplicação, utilizando o **GitBash** e executar o comando conforme instrução abaixo em **Run**.

**Obs.:** No arquivo "**docker-compose.yml**" contém toda a configuração e o script de **CREATE TABLE** necessário para execução da aplicação.

## Run

```sh
$ docker-compose up
```
